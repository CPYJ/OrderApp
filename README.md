
## **🖥️**프로젝트 소개

- 도서를 주문할 수 있는 웹 서비스 입니다.
  <br>회원 CRUD, 상품 CRUD, 주문 CRD 기능이 포함되어 있습니다.
  
<br>  

### 개발환경

- Java 11

- **IDE** : Intelli J

- **FrameWork** : SpringBoot(2.4.1)

- **DataBase** : H2

- **ORM** : JPA
  
<br>
